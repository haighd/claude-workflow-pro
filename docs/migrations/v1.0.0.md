## Installation

### Prerequisites

- **Claude Code CLI**: Latest version
- **Git**: v2.30+
- **Node.js**: v16+ (for certain integrations)
- **OS**: macOS, Linux, or Windows with WSL

### Fresh Installation

1. **Clone the Repository**

```bash
git clone https://github.com/anthropics/claude-workflow-pro.git
cd claude-workflow-pro
```

2. **Verify Installation**

```bash
# Check all commands are available
ls commands/workflow/
ls commands/github/
ls commands/team/
ls agents/
ls templates/
```

3. **Configure Settings** (Optional)

```bash
# Copy example configuration
cp .claude-settings.example.json .claude-settings.json

# Edit configuration
vim .claude-settings.json
```

4. **Run Tests** (Optional but Recommended)

```bash
# Run basic workflow tests
./tests/integration/workflows/test-basic-workflow.sh

# Run all tests
for test in tests/integration/{workflows,agents,examples}/test-*.sh; do
  echo "Running $test..."
  $test
done
```

### Configuration

#### Global Settings

Create `~/.config/claude-code/settings.json`:

```json
{
  "workflow": {
    "checkpoint": {
      "directory": "docs/sessions",
      "autoSave": true,
      "interval": 300
    },
    "execution": {
      "maxSteps": 100,
      "timeout": 3600,
      "retries": 3
    },
    "validation": {
      "strict": true,
      "requireTests": true
    }
  },
  "github": {
    "autoReview": {
      "enabled": true,
      "minScore": 7.0
    },
    "autoTriage": {
      "enabled": true,
      "autoLabel": true
    }
  },
  "team": {
    "pairProgramming": {
      "syncInterval": 5,
      "conflictResolution": "prompt"
    }
  }
}
```

#### Project Settings

Create `.claude-settings.json` in your project:

```json
{
  "workflow": {
    "templates": {
      "default": "feature-development"
    },
    "agents": {
      "enabled": ["planner", "validator", "executor"],
      "disabled": []
    }
  },
  "metrics": {
    "enabled": true,
    "collect": ["workflow", "code", "team", "quality"],
    "exportFormat": "json"
  },
  "integrations": {
    "github": true,
    "slack": false,
    "jira": false
  }
}
```

## Troubleshooting

### Common Issues

#### Issue: Command not found

```bash
# Symptom
/workflow/init
# Error: command not found

# Solution
# Ensure you're in the claude-workflow-pro directory
cd /path/to/claude-workflow-pro

# Or add to Claude Code settings
```

#### Issue: Permission denied

```bash
# Symptom
./tests/integration/test-basic-workflow.sh
# Error: Permission denied

# Solution
chmod +x tests/integration/**/*.sh
```

#### Issue: Checkpoint not loading

```bash
# Symptom
/workflow/resume-work @docs/sessions/checkpoint.md
# Error: Checkpoint file not found

# Solution
# Check file path is correct
ls docs/sessions/
# Use tab completion or full path
/workflow/resume-work @docs/sessions/checkpoint-2025-11-16-14-30.md
```
