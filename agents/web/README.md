# Web Research Agents

## Available Agents

### `web-search-researcher.md` - External Resource Research

**Purpose:** Search and analyze external documentation, libraries, and best practices
**Best for:** Finding solutions, researching frameworks, gathering external knowledge

**Capabilities:**

- Searching technical documentation
- Finding implementation examples
- Researching best practices
- Analyzing library documentation

**Result limits:** Returns top 5 most relevant resources
**Output format:** Summarized findings with source links

## Usage Guidelines

1. Specify clear search terms and context
2. Focus on authoritative sources
3. Cross-reference with internal implementation

## Efficiency Tips

- Provide specific version requirements
- Focus searches on official documentation
- Request summaries of key points only
